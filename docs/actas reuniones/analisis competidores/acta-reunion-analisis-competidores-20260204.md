### 4. Consenso en la forma de realizar las diapositivas
Se discutió cómo se realizarían las diapositivas y se analizarían los competidores y:  

- Se concluyó que se incluirían las características principales de las aplicaciones, las diferencias clave con respecto a nuestra aplicación, y se mencionaría la cuota y modelo de negocio.  

- En el caso de añadir información adicional a la encontrada en la diapositiva, se añadiría en las Notas del orador. 

- Se concluyó que se crearía una diapositiva que mostrara los nombres de todos los competidores encontrados.
