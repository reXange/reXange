### 1. Tecnologías a utilizar

### 2. Diseño básico (esquema de pantalla principal, menús principales, navegación entre casos de uso core)
Despues, hemos definido las pantallas que han de hacerse. Las pantallas quedarían definidas mediante una lista de la siguiente manera:

- Home(): En la pantalla de inicio se mostrarán directamente algunas ofertas relevantes según los datos del usuario. 
- Buscador (CU04, CU05): Se mostrarán los artículos que cumplan con los parámetros de búsqueda y filtros.
- Detalles Usuario (CU09, CU13, CU19, CU20): Se mostrará la información de un usuario en concreto. 
- Editar Usuario (CU17): Se mostrará la información a editar del usuario que esta logueado.
- Chat (CU07, CU08, CU12): Se mostrarán los mensajes enviados entre dos usuarios.
- Detalles Oferta (CU03, CU06, CU14, CU13, CU16, CU18): Se mostrarán los detalles de una oferta en concreto. 
- Editar Oferta (CU02): Se mostrarán la información a editar de la oferta seleccionada. 
- Login y Registro (CU10, CU11).
- Crear Oferta (CU01): Se mostrará un form para crear una oferta.
- Admin Usuarios(CU21): Listado de todos los usuarios. 
- Admin Ofertas(CU22): Listado de todas las ofertas. 
- Admin Valoraciones(CU23): Listado de todas las valoraciones.

PLANTILLA BASE:
